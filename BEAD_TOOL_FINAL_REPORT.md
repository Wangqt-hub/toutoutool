## 🚀 如何使用

### 快速开始

```bash
# 1. 安装依赖（如果还没安装）
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问拼豆工具
http://localhost:3000/tools/bead
```

### 使用流程

#### 模式 1：图片转像素画 ✅

1. **选择模式** - 点击"图片转像素画"卡片
2. **上传图片** - 点击或拖拽图片到上传区域
3. **设置参数**：
   - 网格大小：32×32（推荐新手）
   - 颜色数量：24 色（平衡选择）
   - 勾选"保持宽高比"
   - 选择你的豆子品牌
4. **生成图纸** - 点击按钮
5. **查看结果**：
   - 预览图纸效果
   - 查看豆子统计
   - 导出 CSV 清单

#### 进入拼豆编辑器 ✅

生成图纸后，滚动到页面底部即可看到**拼豆编辑器**：

1. **缩放和拖拽**：
   - 使用工具栏按钮或滚轮缩放
   - 拖拽画布查看细节
   
2. **标记进度**：
   - 点击格子选中某种颜色
   - 右键或长按标记为已完成
   - 查看进度条变化

3. **导出和保存**：
   - 点击"导出 CSV"下载数据
   - 保存到云端（需配置 Supabase）
